Startup Funding Data Cleaning

This project explores Indian startup funding using an interactive Power BI dashboard.
The dataset covers details about startups, locations, industries, investors, funding amounts, and dates.



Data Preparation:

Cleaned and transformed the dataset in Power Query

Removed missing values, corrected city names, and fixed inconsistent text

Converted funding amounts into proper numeric fields

Added new columns like Year, Month, and cleaned the Investors field

Eliminated duplicate records


 Dashboard Overview

The dashboard includes:

Total Funding, Startup Count, Investor Count

Industry-wise funding (Bar Chart)

Funding trend by year (Line Chart)

Top funded cities (Column Chart)

Leading investors (Donut Chart)

Detailed table showing all startup information

Slicers for Year, Month, City, Industry, and Investor



Dataset Source

Dataset used from Kaggle:
https://www.kaggle.com/datasets/sudalairajkumar/indian-startup-funding




Tools Used

Power BI

Power Query

Excel (for quick validation)




Goal of the Project

To understand:

Which industries attract the highest funding

Which cities receive the most investments

Who the major investors are

How startup funding trends evolve over the years
